Au cours de cette formation nous allons voir 2 matières :

Développement Front

Le HTML :

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, ect.

Le Javascript

Javascript est un langage de programmation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramètres

ReactJS :

React est un bibliothèque Javascript libre dont le but est de faciliter la création d'application web monopage, via la création de composants d'interface dynamiques.

Développement Back

Le PHP :

Le PHP est un langage de programmation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certaines informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur ect).

Node :

Node.js est une plateforme logicielle libre et événementielle en JavaScript orientée vers les applications réseau qui doivent pourvoir monter en charge.

Python :

Python est un langage de programmation interprété, multi-paradigme et multiplateformes. Il favorise la programmation impérative structurée, fonctionelle et orientée objet.

630 heures de formation :

Le programme de développement de la prmière année chez HETIC comprends les matières suivantes :

  • Bases de données (SQL et NoSQL) niveau 1
  • API niveau 1
  • Développement Front -HTML/CSS
  • Développement Front -JS Vanilla
  • Développement Front -ReactJS
  • Développement Back - PHP et framework

  • Développement Back - Python niveau 1
